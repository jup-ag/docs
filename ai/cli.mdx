The Jupiter CLI (`jup`) lets you interact with Jupiter's products from the command line. Trade spot, open perps positions, earn yield, and more.

<Warning>
The CLI is pre-v1 (early alpha) and should be considered unstable. Breaking changes may be introduced without warning.
</Warning>

## Install

<Tabs>
  <Tab title="npm">
  ```bash
  npm i -g @jup-ag/cli
  ```
  </Tab>
  <Tab title="Install script">
  Auto-detects the best install method:
  ```bash
  curl -fsSL https://raw.githubusercontent.com/jup-ag/cli/main/scripts/install.sh | bash
  ```
  </Tab>
  <Tab title="Standalone binary">
  Download the latest binary from [GitHub releases](https://github.com/jup-ag/cli/releases):
  ```bash
  curl -fsSL https://github.com/jup-ag/cli/releases/latest/download/jup-linux-x64 -o jup
  chmod +x jup
  sudo mv jup /usr/local/bin/
  ```
  Replace `jup-linux-x64` with the appropriate binary for your platform (`jup-darwin-arm64`, `jup-darwin-x64`, etc.).
  </Tab>
</Tabs>

## Set Up a Key

A key is required for signing transactions (swaps, transfers, lending). Keys are stored locally at `~/.config/jup/keys/`.

```bash
# Generate a new key
jup keys add mykey

# Or import from a file, private key, or seed phrase
jup keys add mykey --file /path/to/keypair.json
jup keys add mykey --private-key <key>
jup keys add mykey --seed-phrase "word1 word2 ..."
```

Verify your setup:

```bash
jup keys list
jup spot portfolio
```

## Ways to Use

### Terminal

The most direct way. Install, add a key, and start trading:

```bash
# Swap 1 SOL to USDC
jup spot swap --from SOL --to USDC --amount 1

# Open a 3x long SOL position with $10 USDC
jup perps open --asset SOL --side long --amount 10 --input USDC --leverage 3

# Deposit 100 USDC into lending
jup lend earn deposit --token USDC --amount 100
```

### Telegram

Use the CLI through [OpenClaw](https://openclaw.ai), an open-source AI assistant that runs locally and connects to Telegram and other chat apps (WhatsApp, Discord, Slack, Signal).

1. Install OpenClaw:
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```
2. Connect your Telegram account by following the [OpenClaw setup guide](https://openclaw.ai)
3. Install the Jupiter CLI:
   ```bash
   npm i -g @jup-ag/cli
   ```
4. Add a key:
   ```bash
   jup keys add mykey
   ```
5. Message your OpenClaw bot on Telegram:
   > "Swap 1 SOL to USDC"

   OpenClaw executes the CLI command on your behalf and returns the result in chat.

<Tip>
OpenClaw can also run scheduled and automated tasks. Set up recurring swaps, portfolio checks, or position monitoring directly from Telegram.
</Tip>

### AI Agents

The CLI is designed to be LLM-friendly. All commands are non-interactive and support JSON output for structured, parseable responses.

Enable JSON mode globally:
```bash
jup config set --output json
```

Or per-command:
```bash
jup spot swap --from SOL --to USDC --amount 1 -f json
```

AI agents (Claude Code, Cursor, Codex, etc.) can invoke CLI commands directly through shell access. Point your agent at the [CLI documentation on GitHub](https://github.com/jup-ag/cli/tree/main/docs) for complete command references and JSON response schemas.

You can also use the CLI's built-in help:
```bash
jup --help
jup spot --help
jup perps --help
jup lend --help
```

<Tip>
Set an API key from [Portal](https://portal.jup.ag) for higher rate limits when running automated workflows:
```bash
jup config set --api-key <your-key>
```
</Tip>

## Products

### Spot

Search tokens, get quotes, execute swaps, view portfolio, check trade history, and transfer tokens.

```bash
jup spot tokens --search SOL
jup spot quote --from SOL --to USDC --amount 1
jup spot swap --from SOL --to USDC --amount 1
jup spot portfolio
jup spot history
jup spot transfer --token SOL --to <address> --amount 1
```

### Perps

Trade perpetual futures on SOL, BTC, and ETH. Open and close positions, set take-profit and stop-loss, and place limit orders.

```bash
jup perps markets
jup perps open --asset SOL --side long --amount 10 --input USDC --leverage 3
jup perps positions
jup perps set --position <pubkey> --tp 100 --sl 70
jup perps close --position <pubkey>
jup perps history
```

### Lend

View available tokens and APY, deposit to earn yield, and withdraw.

```bash
jup lend earn tokens
jup lend earn deposit --token USDC --amount 100
jup lend earn positions
jup lend earn withdraw --token USDC --amount 50
```
