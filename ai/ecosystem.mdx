## Agent Frameworks

Frameworks for building autonomous AI agents that interact with Solana protocols.

<Card title="Solana Agent Kit" icon="robot" href="https://github.com/sendaifun/solana-agent-kit">
  **SendAI** — Open-source toolkit connecting AI agents to 30+ Solana protocols with 60+ actions. Plugin-based: `@solana-agent-kit/plugin-defi` includes Jupiter swaps out of the box.
</Card>
<Card title="Eliza" icon="message-bot" href="https://github.com/elizaOS/eliza">
  **elizaOS** — Lightweight TypeScript AI agent framework with Solana integrations. Popular for building social agents (Twitter/X, Discord, Telegram) that can trade.
</Card>
<Card title="GOAT" icon="link" href="https://github.com/goat-sdk/goat">
  **Crossmint** — Open-source toolkit for connecting AI agents to 200+ onchain tools across multiple chains including Solana.
</Card>
