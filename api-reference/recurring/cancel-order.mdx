<Warning>
The Recurring API is no longer actively maintained and has been superseded by [DCA](/trigger/dca).
</Warning>

<Info>
  **NOTE**

  - `recurringType` is used to denote the type of recurring order, only `time`
  - **DEPRECATED**: `recurringType: price` based orders are deprecated
  - Refer to [Recurring API doc](/recurring/cancel-order) for more information.
</Info>
