## V2 (Recommended)

The unified Swap API. Two paths: `/order` for managed execution, `/build` for custom transactions.

**Base URL:** `https://api.jup.ag/swap/v2`

<CardGroup>
  <Card title="GET /order" href="/api-reference/swap/v2/order" icon="file-lines" horizontal>
    Get a swap quote and assembled transaction with managed execution
  </Card>
  <Card title="GET /build" href="/api-reference/swap/v2/build" icon="file-lines" horizontal>
    Get a swap quote with raw instructions for building custom transactions
  </Card>
  <Card title="POST /execute" href="/api-reference/swap/v2/execute" icon="file-lines" horizontal>
    Execute a signed transaction from /order with managed landing
  </Card>
</CardGroup>

## V1 (Metis)

The original low-level routing engine for maximum flexibility and composability.

**Base URL:** `https://api.jup.ag/swap/v1`

<CardGroup>
  <Card title="GET /quote" href="/api-reference/swap/v1/quote" icon="file-lines" horizontal>
    Request a swap quote
  </Card>
  <Card title="POST /swap" href="/api-reference/swap/v1/swap" icon="file-lines" horizontal>
    Build a serialised swap transaction from a quote
  </Card>
  <Card title="POST /swap-instructions" href="/api-reference/swap/v1/swap-instructions" icon="file-lines" horizontal>
    Get raw swap instructions from a quote
  </Card>
  <Card title="GET /program-id-to-label" href="/api-reference/swap/v1/program-id-to-label" icon="file-lines" horizontal>
    Map program IDs to human-readable labels
  </Card>
</CardGroup>
