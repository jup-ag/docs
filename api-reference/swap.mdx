The unified Swap API. Two paths: `/order` for managed execution, `/build` for custom transactions.

<CardGroup>
  <Card title="GET /order" href="/api-reference/swap/order" icon="file-lines" horizontal>
    Get a swap quote and assembled transaction with managed execution
  </Card>
  <Card title="GET /build" href="/api-reference/swap/build" icon="file-lines" horizontal>
    Get a swap quote with raw instructions for building custom transactions
  </Card>
  <Card title="POST /execute" href="/api-reference/swap/execute" icon="file-lines" horizontal>
    Execute a signed transaction from /order with managed landing
  </Card>
</CardGroup>

**Base URL:** `https://api.jup.ag/swap/v2`

All endpoints require an API key via the `x-api-key` header. Get one at [portal.jup.ag](https://portal.jup.ag).
