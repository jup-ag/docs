## Endpoints

<CardGroup>
  <Card title="GET /quote" href="/api-reference/swap/v1/quote" icon="file-lines" horizontal>
    Request a swap quote
  </Card>
  <Card title="POST /swap" href="/api-reference/swap/v1/swap" icon="file-lines" horizontal>
    Build a serialised swap transaction from a quote
  </Card>
  <Card title="POST /swap-instructions" href="/api-reference/swap/v1/swap-instructions" icon="file-lines" horizontal>
    Get raw swap instructions from a quote
  </Card>
  <Card title="GET /program-id-to-label" href="/api-reference/swap/v1/program-id-to-label" icon="file-lines" horizontal>
    Map program IDs to human-readable labels
  </Card>
</CardGroup>

**Base URL:** `https://api.jup.ag/swap/v1`
