Endpoints: GET /search (search by symbol, name, or mint), GET /tag (tokens by tag), GET /{category}/{interval} (tokens by category), GET /recent (recently created pools), GET /content (approved token content), GET /content/cooking (trending token content), GET /content/feed (paginated content feed). Base URL: api.jup.ag/tokens/v2."
