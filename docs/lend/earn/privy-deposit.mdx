Deposit assets into Jupiter Earn vaults using Privy embedded wallets. Users can earn yield without managing their own private keys.

---

## Deposit to Earn with Privy

<Steps>
  <Step>
    ### Import Dependencies

    Import the required packages for Privy client, Solana RPC, and Jupiter Lend Earn SDK operations.

    ```typescript
    import { PrivyClient } from "@privy-io/node";
    import {
      Connection,
      PublicKey,
      TransactionMessage,
      VersionedTransaction,
    } from "@solana/web3.js";
    import BN from "bn.js";
    import { getDepositIxs } from "@jup-ag/lend/earn";
    ```
  </Step>

  <Step>
    ### Configure Privy and Initialize Client

    Set up your Privy credentials and Solana connection.

    ```typescript
    const PRIVY_APP_ID = "put-your-privy-app-id-here";
    const PRIVY_APP_SECRET = "put-your-privy-app-secret-here";
    const RPC_ENDPOINT = "https://api.mainnet-beta.solana.com";
    const ASSET_MINT = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"; // USDC
    const DEPOSIT_AMOUNT = new BN(10_000_000); // 10 USDC (6 decimals)
    const AUTH_KEY_ID = "put-your-auth-key-id-here";
    const AUTH_KEY_PRIVATE = "put-your-auth-key-private-here";
    const WALLET_ID = "put-your-wallet-id-here";

    const privy = new PrivyClient({
      appId: PRIVY_APP_ID,
      appSecret: PRIVY_APP_SECRET,
    });

    const connection = new Connection(RPC_ENDPOINT);
    ```

    <Note>
      Update the configuration values with your Privy app credentials. Ensure the Privy wallet holds SOL (for fees) and USDC (for deposit).
    </Note>
  </Step>

  <Step>
    ### Get Wallet Address

    Retrieve the Privy wallet to obtain the signer public key.

    ```typescript
    const wallet = await privy.wallets().get(WALLET_ID);
    const signer = new PublicKey(wallet.address);
    ```
  </Step>

  <Step>
    ### Build Deposit Instructions

    Use the Jupiter Lend Earn SDK to generate deposit instructions.

    ```typescript
    const { ixs: depositIxs } = await getDepositIxs({
      connection,
      signer,
      asset: new PublicKey(ASSET_MINT),
      amount: DEPOSIT_AMOUNT,
    });

    if (!depositIxs?.length) {
      throw new Error("No deposit instructions returned by Jupiter Lend SDK");
    }
    ```
  </Step>

  <Step>
    ### Build Transaction Message

    Build an unsigned transaction that Privy will sign.

    ```typescript
    const { blockhash, lastValidBlockHeight } =
      await connection.getLatestBlockhash("finalized");

    const message = new TransactionMessage({
      payerKey: signer,
      recentBlockhash: blockhash,
      instructions: depositIxs,
    }).compileToV0Message();

    const transaction = new VersionedTransaction(message);
    const serializedTx = Buffer.from(transaction.serialize()).toString("base64");
    ```
  </Step>

  <Step>
    ### Sign Transaction with Privy

    Send the serialized transaction to Privy for signing.

    ```typescript
    const signResponse = await privy
      .wallets()
      .solana()
      .signTransaction(WALLET_ID, {
        transaction: serializedTx,
        authorization_context: {
          authorization_private_keys: [AUTH_KEY_PRIVATE],
        },
      });
    ```

    <Info>
      Privy signs the transaction on behalf of the embedded wallet. The private key never leaves Privy's infrastructure.
    </Info>
  </Step>

  <Step>
    ### Send Transaction

    Submit the signed transaction to the Solana network.

    ```typescript
    const signedTxBytes = Buffer.from(
      signResponse.signed_transaction as string,
      "base64"
    );
    const signature = await connection.sendRawTransaction(signedTxBytes, {
      skipPreflight: true,
    });
    await connection.confirmTransaction(
      { signature, blockhash, lastValidBlockHeight },
      "confirmed"
    );
    console.log("Deposit successful! Signature:", signature);
    ```

    <Check>
      The deposit is complete. The user's funds are now earning yield in Jupiter Earn via their Privy embedded wallet.
    </Check>
  </Step>
</Steps>
