The two Swap API paths have different fee models. This page covers both.

## Fee comparison

| | `/order` | `/build` |
| --- | --- | --- |
| **Jupiter swap fee** | Yes (platform fee) | **No** |
| **Integrator fees** | Referral fees or advanced fees (feeBps/feeFlat) | Platform fee only (platformFeeBps) or DIY |
| **RFQ routing** | Disabled when fee params are added | Not available |

## `/order` fees

### Jupiter platform fee

Jupiter charges a platform fee on `/order` swaps. This fee is included in the quote and deducted automatically. The `platformFee` field in the response shows the fee amount and rate:

```json
{
  "platformFee": {
    "amount": "8529",
    "feeBps": 5,
    "feeMint": "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
  }
}
```

### Referral fees

Use the [Jupiter Referral Program](/tool-kits/referral-program) to earn fees on swaps. Set `referralAccount` and `referralFee`:

```typescript
const params = new URLSearchParams({
  inputMint: "So11111111111111111111111111111111111111112",
  outputMint: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  amount: "1000000000",
  taker: walletAddress,
  referralAccount: "YOUR_REFERRAL_ACCOUNT",
  referralFee: "50", // 0.5%
});

const response = await fetch(
  `https://api.jup.ag/swap/v2/order?${params}`,
  { headers: { "x-api-key": API_KEY } }
);
```

<Warning>
  Adding `referralAccount` disables RFQ routing. You will only get Metis quotes.
</Warning>

### Advanced fees (feeBps / feeFlat)

For more control over fee collection, use the advanced fee parameters:

| Parameter | Description |
| --- | --- |
| `feeBps` | Fee as a portion of swap input USD value (1-10000 bps). ExactIn only. |
| `feeFlat` | Flat fee amount in `feeMint`. |
| `feeReducer` | Required if both feeBps and feeFlat set. `min` (lower of the two), `max` (higher), or `sum` (both combined). |
| `feeMint` | Token to collect fees in. Required. |
| `feeReceiver` | Wallet to receive fees. Required. |

```typescript
const params = new URLSearchParams({
  inputMint: "So11111111111111111111111111111111111111112",
  outputMint: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  amount: "1000000000",
  taker: walletAddress,
  feeBps: "100",     // 1%
  feeMint: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  feeReceiver: "YOUR_WALLET_ADDRESS",
});
```

The response `advancedFee` field shows the calculated fee:

```json
{
  "advancedFee": {
    "feeBps": "100",
    "feeReceiver": "YOUR_WALLET_ADDRESS",
    "feeMint": "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
    "feeUsdValue": 1.71
  }
}
```

<Warning>
  Advanced fees also disable RFQ routing, restricting to Metis-only quotes.
</Warning>

## `/build` fees

Jupiter does not charge swap fees on `/build`. The only fee mechanism is integrator platform fees via `platformFeeBps` and `feeAccount`:

```typescript
const params = new URLSearchParams({
  inputMint: "So11111111111111111111111111111111111111112",
  outputMint: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  amount: "1000000000",
  userPublicKey: walletAddress,
  platformFeeBps: "50",          // 0.5%
  feeAccount: "YOUR_FEE_TOKEN_ACCOUNT",
});

const response = await fetch(
  `https://api.jup.ag/swap/v2/build?${params}`,
  { headers: { "x-api-key": API_KEY } }
);
```

The platform fee is collected as part of the swap instruction. You must provide a token account (`feeAccount`) to receive the fees.

## Fee response fields

The `/order` response includes these fee-related fields:

| Field | Description |
| --- | --- |
| `feeBps` | Total fees including platform and other fees (basis points) |
| `feeMint` | Token fees are collected in |
| `platformFee.amount` | Jupiter platform fee amount |
| `platformFee.feeBps` | Jupiter platform fee rate |
| `advancedFee` | Integrator fee details (present if feeBps/feeFlat used) |
