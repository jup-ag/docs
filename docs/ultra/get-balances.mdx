<Warning>
The Ultra Swap API has been superseded by the [Swap API V2](/docs/swap). Use [Order & Execute](/docs/swap/order-and-execute) for the recommended swap flow.
</Warning>

**Deprecated:** This endpoint returns simplified token balances. Use [Get Holdings](/docs/ultra/get-holdings) (`/ultra/v1/holdings/{address}`) instead for detailed token account information including frozen status and ATA flags.

Fetch balances for a wallet (deprecated):

```js
const balancesResponse = await (
  await fetch(`https://api.jup.ag/ultra/v1/balances/3X2LFoTQecbpqCR7G5tL1kczqBKurjKPHhKSZrJ4wgWc`,
    {
      headers: {
        'x-api-key': 'your-api-key',
      },
    }
  )
).json();

console.log(JSON.stringify(balancesResponse, null, 2));
```

## Balances Response

The balances response will return a list of token balances for the user's wallet address.

**Successful example response:**

```json
{
  "SOL": {
    "amount": "0",
    "uiAmount": 0,
    "slot": 324307186,
    "isFrozen": false
  }
}
```

**Failed example response:**

```json
{
  "error": "Invalid address"
}
```
