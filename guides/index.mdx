## What are these guides?

These guides answer the question: **"How do I do X on Solana?"**

Each guide covers one developer intent, explains which Jupiter API to use, and provides working code you can copy directly into your project.

**Before you start:** Get an API key at [portal.jup.ag](https://portal.jup.ag)
