## Labelling

When integrating with Jupiter products, you are advised to correctly label the APIs used.

<Warning>

    | API Used                                             | Required Label           |
    | ---------------------------------------------------- | ------------------------ |
    | Ultra Swap API                                       | Ultra                    |
    | Metis Swap API                                       | Metis                    |
    | Self-Hosted Metis Swap API binary                    | Metis                    |
</Warning>
